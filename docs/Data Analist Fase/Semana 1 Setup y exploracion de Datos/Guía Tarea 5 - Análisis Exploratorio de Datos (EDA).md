### **3. Interacciones y Patrones Ocultos**

**Pregunta 3.1:** ¬øQu√© porcentaje de pel√≠culas en el dataset tiene menos de 5 calificaciones? ¬øQu√© implicaciones tiene esto para un modelo de filtrado colaborativo?

Esto tiene¬†**implicaciones muy serias**¬†para un modelo de filtrado colaborativo:

1. **Falta de Se√±al Suficiente:**¬†El filtrado colaborativo funciona encontrando patrones en c√≥mo los usuarios califican las pel√≠culas. 3. **Riesgo de Sobreajuste (Overfitting):**¬†El modelo podr√≠a darle demasiada importancia a las pocas calificaciones que existen. Por ejemplo, si una pel√≠cula tiene una sola calificaci√≥n de 5 estrellas, el modelo podr√≠a pensar err√≥neamente que es una obra maestra universal.
    

**Conclusi√≥n Pr√°ctica:**¬†Para entrenar nuestro primer modelo, probablemente deber√≠amos considerar¬†**excluir estas pel√≠culas**¬†o tratarlas de una manera especial, ya que no proporcionan suficiente informaci√≥n fiable para el algoritmo.
